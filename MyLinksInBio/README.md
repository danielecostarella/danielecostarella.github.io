# MyLinksInBio

A minimal "links in bio" page — the single URL I put in my social profiles to collect all my links in one place.

## Links

| Link | Description |
|---|---|
| [GitHub](https://github.com/danielecostarella) | What I'm coding |
| [Last.fm](https://www.last.fm/it/user/blackbliss) | What I'm listening to |
| [Goodreads](https://goodreads.com/danielecostarella) | What I'm reading |
| [X](https://x.com/dcostarella) | Talk to me on X |

## Stack

Plain HTML + CSS. No framework, no build step.

- Dark theme with yellow accent
- Fully responsive (mobile-first, single column)
- Inline SVG icons — no external icon libraries
- Staggered entrance animations, hover lift effect
- Respects `prefers-reduced-motion`
